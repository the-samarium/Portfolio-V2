# Portfolio

A minimal, fast personal portfolio website focused on clean design, performance, and modular UI components.

## Overview
This project is built as a lightweight showcase website using modern frontend tooling.  
It emphasizes simplicity, responsiveness, and reusable components without unnecessary complexity.

## Tech Stack
- **Next.js** – App Router, optimized routing and rendering  
- **React** – Component-driven UI architecture  
- **Tailwind CSS** – Utility-first styling for rapid and consistent design  
- **shadcn/ui** – Accessible, composable UI components  
- **Aceternity UI** – Motion-based and interactive components  

## Design Goals
- Minimal and professional layout  
- Fast load times and smooth interactions  
- Reusable and maintainable component structure  
- Clean typography and spacing  

## Features
- Responsive layout across devices  
- Modular UI blocks and sections  
- Modern animations with subtle motion  
- Easy to extend and customize
